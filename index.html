7. 优化策略路线图

基于以上分析，制定分优先级的可执行优化行动计划。

🔴 P0 - 立即执行：部署账户级负面关键词

将层级1的账户级负面关键词（烹饪教程、信息查询、娱乐内容）批量添加至账户。预计可立即减少约2,266次/月的无效曝光，将零转化查询比例从65%降至50%以下。

预计节省10-15%无效花费

🔴 P0 - 立即执行：排查生蚝品类消失原因

生蚝在12-3月贡献¥5,614营收（68单），但在4-5月营收为¥0。立即检查：(1)商品是否因季节性原因下架；(2)是否有负面关键词误伤了生蚝相关查询；(3)竞价策略是否被暂停或大幅降低。

潜在损失：~¥1,400/月

🟠 P1 - 本周执行：为减脂餐品类建立独立 Campaign

减脂餐/健康在2个月内贡献¥26,191营收，CVR高达9.11%，是增速和价值最高的品类。建议：建立独立的减脂餐 Campaign,添加「减脂餐」「减肥餐」「健康餐」「低脂」等精确关键词；为高频转化词（如"减脂餐食谱""减肥餐食谱"）设置高出价。

可释放 20-30% 额外增长

🟠 P1 - 本周执行：将45个高潜力长尾词批量加入精确匹配

从机会挖掘中识别出的45个高CTR+高CVR长尾关键词，当前曝光量低但转化效率极高。将它们批量添加为精确匹配关键词并设置较高出价，预计可带来额外30-50单/月。

预计新增¥3,500-5,000/月营收

🟠 P1 - 本周执行：优化虾类品类的广告创意

虾类是第三大品类（¥19,882），但CTR仅6.49%（远低于三文鱼的19.07%）。虾的搜索词中大量混合了菜谱类和信息类查询。建议：(1)优化虾类广告标题，强化购买信号（价格、新鲜、包邮）；(2)针对虾仁、大虾等子品类建立独立 Ad Group 做精准匹配。

CTR提升至10%+可增加30%点击

🟡 P2 - 本月执行：构建竞品关键词防御策略

🟡 P2 - 本月执行：建立关键词生命周期监控仪表盘

每月有上百个新搜索词出现，同时有上百个消失。建议：(1)每周review新增高价值搜索词，及时加入精确关键词；(2)跟踪连续两个月零转化的高曝光词，自动化加入负面关键词；(3)监控衰退关键词（曾转化但连续2月消失的），触发库存/竞品/策略检查。

🟢 P3 - 季度执行：搜索词意图路由优化

8.3 匹配类型策略

不同词层采用不同的匹配类型组合，平衡流量覆盖与精准控制。

🎯 精确匹配
适用词层：头词 + 长尾词
  • 头词：精确匹配确保流量质量，避免宽泛匹配带来噪音（如"三文鱼"匹配到"三文鱼热量"）
  • 长尾词：精确匹配锁定高意向用户，争抢精准转化（如"30天减肥食谱表"）
  • 预计覆盖：50-60%关键词
🔍 词组匹配
适用词层：品类词
  • 在保持意图方向的前提下扩展变体（如"减脂餐食谱" → "减脂餐日常食谱"）
  • 搭配负面关键词过滤"怎么做""热量"等信息型后缀
  • 预计覆盖：20-25%关键词
🌐 广泛匹配（管控型）
适用词层：拓展词
  • 仅用于新词发现和竞品截流，必须搭配严密的负面关键词
  • 每周 review 搜索词报告，将有转化的查询升级为精确匹配
  • 预计覆盖：15-20%关键词
⚠️ 核心原则：广泛匹配是"矿工"而非"主力"。它用来发现黄金（新高转化词），但本身消耗大量废石（无效查询）。已有的分析显示：65.2%的搜索词来自广泛匹配的零转化流量。执行P0负面关键词策略后，再谨慎开启广泛匹配进行拓词。
